# TAAG プロジェクト — Claude向け起動指示

このフォルダは **TAAG（武田建築設計事務所 / TAKEDA ARCHITECT & ASSOCIATES）** のウェブサイト構築プロジェクトです。

## 作業開始前の必須手順

1. **`TAAG_開発メモ.md` を最初に必ず読むこと**
   - プロジェクトの全体像、CMS環境、ファイル構成、設計方針、決定事項がすべて記載されています
   - 特に冒頭の「★コードレビュー報告ポリシー（恒久ルール）」は本プロジェクトの全ての回答で遵守すること

2. メモを読んだら、ユーザー（直美さん）に「開発メモを読みました」と一言伝えてから作業に入ること

## コードレビュー報告ポリシー（要約・詳細はメモ参照）

コードを生成・修正したら、回答末尾に必ず以下を報告：
- 外部URL・CDNからの読み込み（script / link / @import / webfont 等）の列挙
- fetch / XMLHttpRequest / WebSocket 等の外部通信があれば送信先と内容
- 該当なしの場合は「外部通信・外部読み込みなし」と明記

## マルチデバイス対応ポリシー（要約・詳細はメモ参照）

サイト用コードはPCとモバイル両方を常に意識：
- viewport meta必須、レスポンシブ前提（%, vw, clamp 活用）
- タッチ操作対応、本文14px以上、横スクロール厳禁
- アニメーション・SVGの負荷も配慮

## 主要ファイル

- `TAAG_開発メモ.md` — プロジェクト全体の知識ベース（最重要）
- `src/` — ソース・オブ・トゥルース（CSS/JS/HTML分離構成、CMS移行に強い）
- `taag_april_entrance.html` — 4月版エントランス（モノリシック旧版・参照用）
- `images/` — 画像素材
